All you would want to know about Backflow Testing

Does my home have a backflow prevention device?

If you have a sprinkler system or a swimming pool you must have a backflow prevention in your home. Several homes have it for other reasons too. It is typically covered in a green case like this one

Is it mandatory to get an annual backflow testing?

The short answer is , “YES”, it is mandatory to get your backflow prevention device certified each year.

Book Backflow Testing Now

Some cities have a fixed date by which a homeowner must get their backflow certified, others like Germantown, TN and Collierville, TN ask you to get it done any time in the year as long as you get it done on an annual basis... Read more »
